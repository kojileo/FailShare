# FailShare アプリコンセプト

## 🎯 アプリの目的

### 基本コンセプト
**「失敗を共有して、みんなで成長する」**

失敗談を匿名で共有し、同じような経験を持つ人たちが支え合い、学び合えるコミュニティプラットフォームです。

### 解決したい問題
1. **失敗への恐怖**: 失敗を恐れて挑戦を諦めてしまう
2. **孤独感**: 失敗した時に一人で抱え込んでしまう
3. **学習機会の損失**: 他人の失敗から学ぶ機会が少ない
4. **成長の停滞**: 失敗から学ぶことで得られる成長機会の損失

### 提供する価値
1. **安心感**: 匿名で失敗を共有できる安全な環境
2. **共感**: 同じような経験を持つ人たちとの繋がり
3. **学習**: 他人の失敗から学ぶ機会

## 🚀 コア機能

### 1. 失敗談の投稿・共有 ✅ **実装済み**
- **匿名投稿**: 身元を隠して安心して投稿
- **構造化テンプレート**: 失敗談を整理して投稿
- **カテゴリ分類**: 恋愛・仕事・その他の分野別分類
- **感情タグ**: 後悔・恥ずかしい・悲しいなどの感情表現
- **編集・削除**: 自分の投稿の管理

### 2. 失敗談の閲覧・検索 ✅ **実装済み**
- **カテゴリ別表示**: 興味のある分野の失敗談を閲覧
- **キーワード検索**: 特定のキーワードで失敗談を検索
- **感情別フィルタ**: 感情タグによる絞り込み
- **時系列表示**: 最新の失敗談を優先表示
- **詳細表示**: 失敗談の詳細情報を表示

### 3. エンゲージメント機能 ✅ **実装済み**

#### いいね機能
- **ハートアニメーション**: 視覚的に分かりやすい共感表現
- **リアルタイム更新**: 即座にいいね数が反映
- **いいね履歴**: 自分がいいねした投稿の管理
- **いいね統計**: ユーザー別のいいね獲得数・与えた数

#### コメント機能
- **リアルタイム更新**: 新規コメントの即座表示
- **編集・削除**: 自分のコメントの管理
- **時系列表示**: コメントの時系列での表示
- **楽観的更新**: UI即座更新 + エラー時復旧

### 4. フレンド・コミュニティ機能 ✅ **実装済み**

#### フレンド機能
- **フレンドリクエスト**: 失敗談を通じて知り合ったユーザーをフレンド追加
- **フレンド承認・拒否**: フレンドリクエストの承認・拒否機能
- **フレンド管理**: フレンドの追加・削除・ブロック機能
- **フレンド検索**: 同じ失敗経験を持つユーザーを検索・発見
- **フレンド一覧**: フレンドの投稿・活動をタイムライン表示
- **フレンド推薦**: 共通の失敗経験に基づくフレンド推薦
- **ユーザーブロック**: 不適切ユーザーのブロック機能

### 5. チャット機能（設計中）
- **1対1チャット**: フレンド間のプライベートメッセージ
- **メッセージ制限**: 無料ユーザーの月間メッセージ数制限
- **リアルタイム更新**: 即座のメッセージ配信
- **オフライン対応**: ネットワーク復旧時の同期
